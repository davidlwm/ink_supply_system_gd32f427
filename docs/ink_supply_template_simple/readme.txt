实验器材:
	GD32F427VET6核心板 
	
实验目的:
	学习GD32模拟IIC的使用,驱动24C02实现EEPROM数据读写.
	
硬件资源:
	1,LED0(连接在PA1),LED1(连接在PA2)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	3,2.8/3.5/4.3寸TFTLCD模块(通过FSMC驱动,FSMC_NE0接LCD片选/A18接RS) 
	4,按键KEY0(PC0)/WK_UP(PA0)
	5,24C02(IIC连接在PB8/PB9上面) 

实验现象:
	本实验通过WK_UP按键来控制24C02的写入，通过另外一个按键KEY0来控制24C02的读取。并在LCD模块上面显示
	相关信息。同时，我们可以通过USMART控制在24C02的任意地址写入和读取数据。 
	  
注意事项: 
	1,本例程在LCD_Init函数里面(在lcd.c),用到了printf,如果不初始化串口1,将导致液晶无法显示!!
